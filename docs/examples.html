Examples

Time for some usage examples now, isn't it? For simplicity reasons lets suppose that we have a predefined data set with a user data base containing username, name, lastname and the money included in the user's account. 

    
const data = [
  {
    id: 1,
    name: 'John',
    lastName: 'Doe',
    username: '@john.doe',
    money: 500
  }, ...
];

Simple view

Lets begin with a simple view of the base table without any additional options.

        
<BootstrapDataTable data={data} >
  <BootstrapDataColumn property={'index'} name={'#'} />
  <BootstrapDataColumn property={'name'} name={'Name'} />
  <BootstrapDataColumn property={'lastName'} name={'Last Name'} />
  <BootstrapDataColumn property={'username'} name={'Username'} />
  <BootstrapDataColumn property={'money'} name={'Money'} />
</BootstrapDataTable>

Bordered view

Adding a border makes it prettier, doesn't it?

        
<BootstrapDataTable data={data} bordered >
  <BootstrapDataColumn property={'index'} name={'#'} />
  <BootstrapDataColumn property={'name'} name={'Name'} />
  <BootstrapDataColumn property={'lastName'} name={'Last Name'} />
  <BootstrapDataColumn property={'username'} name={'Username'} />
  <BootstrapDataColumn property={'money'} name={'Money'} />
</BootstrapDataTable>
